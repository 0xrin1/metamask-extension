# Light Theme Colors

Color design token css variables to be used in the styles for MetaMask UI.

### General UI

#### Background

Background can be used for neutral, generally larger components that act as the background for the rest of the color styles.

<ColorPalette>
  <ColorItem
    title="var(--color-background-default)"
    subtitle="(white000: #FFFFFF) background.default should be used as the default background color for any neutral type components."
    colors={['hsla(0, 0%, 100%, 1)']}
  />
  <ColorItem
    title="var(--color-background-alternative)"
    subtitle="(grey040: #F2F4F6) background.alternative should be used as an alternative background for any neutral type components that require some slight contrast to background.default"
    colors={['hsla(210, 18%, 96%, 1)']}
  />
</ColorPalette>
<br />

#### Text

Text can be used for all general text used as the foreground elements on top of background colors.

<ColorPalette>
  <ColorItem
    title="var(--color-text-default)"
    subtitle="(grey800: #24272A) text.default should be used for all general text used on background.default or background.alternative that takes main priority in the information hierarchy."
    colors={['hsla(210, 8%, 15%, 1)']}
  />
  <ColorItem
    title="var(--color-text-alternative)"
    subtitle="(grey600: #535A61) text.alternative should be used for all general text used on background.default or background.alternative that takes less priority in the information hierarchy than text.default"
    colors={['hsla(210, 8%, 35%, 1)']}
  />
  <ColorItem
    title="var(--color-text-muted)"
    subtitle="(grey200: #BBC0C5) text.muted should be used for all low priority text used on background.default or background.alternative. These could also include placeholder or inactive text. It will also not meet AAA or AA accessibility standards."
    colors={['hsla(210, 8%, 75%, 1)']}
  />
</ColorPalette>
